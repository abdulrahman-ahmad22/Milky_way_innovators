# مجموعة المجلس لريادة الأعمال — موقع تجريبي

موقع حكومي الطابع (بسيط ومهني على نمط u.ae) مبني بـ HTML / CSS / JS فقط، مع مساعد ذكي
«رائد أعمال المجالس» متصل بواجهة Google Gemini ويتحدث باللهجة الإماراتية.

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو شغّل خادماً محلياً:

```bash
cd majlis-site
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## المحتوى
- **الرئيسية** — التعريف، الرؤية، الرسالة، والتحديات.
- **الورش القادمة** — أربع ورش تجريبية (بيانات وهمية لأغراض العرض).
- **مساعد الأعمال الذكي** — «رائد أعمال المجالس» (Gemini)، يستند إلى قانون المعاملات
  التجارية الاتحادي رقم (50) لسنة 2022 ويتحدث باللهجة المحلية.
- **نتائج الأبحاث** — مقالات توضيحية مستخلصة من الورش (نماذج).

## الملفات
- `index.html` — الصفحة والمحتوى.
- `styles.css` — التصميم.
- `script.js` — التنقل + ربط مساعد Gemini (النموذج: `gemini-2.5-flash`).
- `assets/majlis-exterior.jpeg` — خلفية الواجهة الرئيسية (مبنى المجلس).
- `assets/majlis-interior.jpeg` — خلفية قسم التحديات (داخل المجلس).

## ملاحظات
- مفتاح Gemini مضمّن في `script.js` (`GEMINI_API_KEY`) كما هو مطلوب للنموذج التجريبي.
  بما أنه مفتاح من جهة العميل فهو مكشوف في المتصفح؛ للإنتاج يُفضّل تمريره عبر خادم وسيط.
- نموذج `gemini-2.0-flash` يعطي خطأ حصة (quota: 0) على هذا المفتاح، لذا استُخدم `gemini-2.5-flash` الذي يعمل.
